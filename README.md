### How to start

**Build website files**

```
cd src/ZKWeb.MVVMDemo.Plugins/MVVM.Angular.Website/static
npm install
npm run build:watch
```

**Start website**

Open ZKWeb.MVVMDemo.sln with VS2017 and just run the project.
