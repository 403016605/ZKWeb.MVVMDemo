### How to start

**Build website files**

```
cd src/ZKWeb.MVVMPlugins/MVVM.Angular.Website/static
cnpm install --global gulp-cli
cnpm install
cnpm install --dev
gulp
```

**Start website**

Open ZKWeb.MVVMDemo.sln with VS2017 and just run the project.
